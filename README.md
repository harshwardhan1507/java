# ☕ Java — Language Practice & Fundamentals

A structured collection of Java programs focused on building strong language fundamentals. This repo covers core concepts, syntax, and OOP principles — laying the groundwork before tackling DSA.

> 🔗 For DSA & LeetCode solutions, check out my [leetcode](https://github.com/harshwardhan1507/leetcode) repo.

---

## 📂 What's Inside

- Variables, Data Types & Operators
- Control Flow (if/else, loops, switch)
- Methods & Recursion
- Object-Oriented Programming (Classes, Objects, Inheritance, Polymorphism, Encapsulation, Abstraction)
- Arrays & Strings
- Exception Handling
- Collections (ArrayList, HashMap, etc.)
- File I/O basics
- Mini practice programs

> More topics will be added as I progress.

---

## 🎯 Goal

Get comfortable with Java as a language — syntax, OOP, and standard library — before moving into Data Structures & Algorithms.

---

## 🛠 How to Run

Make sure you have **JDK 17+** installed.

```bash
# Compile
javac FileName.java

# Run
java FileName
```
